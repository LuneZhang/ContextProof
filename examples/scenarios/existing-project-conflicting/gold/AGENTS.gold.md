# AGENTS.md

## Scope

These instructions apply to frontend work in this repository.

## Working Rules

- Keep changes scoped to the requested task.
- Ask before changing files only when the request, ownership, or safety boundary is unclear.
- Do not ask for permission for routine edits that are already requested by the user.

## Validation

- Frontend changes: `npm test`.
- If tests cannot be run, report the reason and what was checked instead.
